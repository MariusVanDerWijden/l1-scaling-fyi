# Ethereum L1 Scaling Dashboard

A static web dashboard for Ethereum L1 scaling: gas limit metrics, blob scaling, roadblocks, and FAQ. Use the **L1 Gas Limit** and **Blobs** tabs to switch between views. Data sources: `data_gaslimit.json` and `data_blobs.json`.

## Running Locally

```python
python3 -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.
